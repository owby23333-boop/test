# 🔄 网关配对说明

**问题：** 网关绑定为 loopback 模式，需要配对才能执行本地命令（如 npm install、git clone 等）

- **状态：** 需要配对

## 💡 解决方案

### 方案 1：Web Chat 配对（推荐）

通过 Web Chat 界面进行配对，这是最简单的方案：

1. **打开浏览器** - 访问 OpenClaw Web Chat 界面
2. **找到配对选项** - 在设置或配置页面
3. **生成 QR 码** - 网关会显示配对码
4. **扫描配对** - 使用配套应用扫描 QR 码
5. **完成配对** - 等待网关重启并确认状态

### 方案 2：手动修改配置

如果不需要移动端配对，可以直接禁用配对要求：

```json
{
  "gateway": {
    "bind": "all",  // 改为允许所有连接
    "pairing": {
      "required": false  // 不需要配对
    }
  }
}
```

然后重启网关：
```bash
openclaw gateway restart
```

### 方案 3：使用远程连接

如果网关绑定为 loopback，可以：
1. 使用 SSH 隧道访问
2. 或者配置 gateway.remote.url 进行远程连接
3. 然后在另一台设备上配对

## ⚠️ 安全提示

- 确保网络环境安全后再配对
- 配对后会显示二维码
- 配对是一次性的，完成后保持连接
- 如果不需要移动端访问，建议改为 `bind: "all"`
