# 📖 小说阅读器 Web 版 - 使用指南

## 🎯 快速开始

### 1️⃣ 启动应用

打开命令行（PowerShell 或 CMD）：

```bash
cd PROJECT/小说阅读器-Web
npm run dev
```

看到提示后，打开浏览器访问：

```
http://localhost:5173
```

---

## 📝 页面说明

### 📚 书架页面（首页）

#### 功能
- 显示所有已添加的书籍
- 上传新书籍
- 删除书籍

#### 操作
1. **上传书籍**
   - 点击"📥 上传书籍"按钮
   - 选择 txt/md/epub 文件
   - 等待上传完成

2. **开始阅读**
   - 点击书籍卡片上的"开始阅读"
   - 或点击"📚 书名"直接打开

3. **删除书籍**
   - 点击"删除"按钮

---

### 📖 阅读页面

#### 功能
- 阅读书籍内容
- TTS 朗读
- 调节字体大小
- 调节行间距
- 章节跳转

#### 操作

**调节字体：**
- 拖动滑块（12-24px）
- 数值越大，字越大

**调节行间距：**
- 选择下拉菜单
- 1=单倍，2=2 倍

**TTS 朗读：**
- 点击"🔊 朗读"开始
- 点击"⏸️ 暂停"暂停
- 再次点击继续

**章节跳转：**
- "⏮️ 上一章" - 上一步
- "⏭️ 下一章" - 下一步

**返回书架：**
- 点击"← 返回"按钮

---

### ⚙️ 设置页面

#### 功能
- 切换主题（亮色/暗色/护眼）
- 设置默认字体
- 设置默认字体大小
- 设置默认行间距

#### 操作
1. **主题切换**
   - 点击导航栏的"🌓 切换主题"
   - 点击多次可循环切换

2. **默认设置**
   - 在设置页面修改
   - 点击"保存设置"

## 🎮 完整操作流程

### 场景 1：第一次使用

```
1. 启动应用 → npm run dev
2. 打开浏览器 → http://localhost:5173
3. 点击"📥 上传书籍"
4. 选择文件 → 等待上传
5. 看到书籍列表
```

### 场景 2：上传一本新小说

```
1. 在书架页面
2. 点击"📥 上传书籍"
3. 打开文件管理器
4. 选择小说文件（.txt/.md）
5. 点击"打开"
6. 书籍自动添加到书架
```

### 场景 3：阅读小说

```
1. 在书架找到要读的小说
2. 点击"开始阅读"
3. 调节字体大小（拖动滑块）
4. 调节行间距（下拉菜单）
5. 点击"TTS 朗读"听书
6. 使用"上一章/下一章"跳转
```

### 场景 4：删除不需要的书

```
1. 找到要删除的书籍
2. 点击"删除"按钮
3. 确认删除
```

### 场景 5：切换主题

```
方法 1：点击导航栏的"🌓 切换主题"
方法 2：在设置页面选择主题
```

## 💡 使用技巧

### 1. 批量上传
- 多次点击上传按钮
- 每本书独立上传
- 每本书保持独立进度

### 2. 保存进度
- 关闭页面后进度自动保存
- 下次打开继续阅读

### 3. TTS 设置
- 确保系统已安装中文语音包
- Windows 默认语音可用

### 4. 快捷键
- F11 - 全屏阅读
- Ctrl+S - 保存当前设置（待实现）

## 🐛 常见问题

### Q1：点击上传没反应？
A：确认文件是 .txt/.md 格式

### Q2：朗读没声音？
A：检查系统音量、TTS 设置

### Q3：字体调节没效果？
A：刷新页面后生效

### Q4：如何添加本地文件？
A：将文件放在浏览器可访问的目录
或使用文件上传功能

### Q5：书籍进度怎么保存？
A：自动保存在 localStorage
刷新页面后进度保留

## 🎯 最佳实践

### 1. 书籍命名
使用清晰的文件名：
```
书名_作者.txt
```

### 2. 字体大小
- 电脑阅读：16-18px
- 手机阅读：14-16px

### 3. 行间距
- 密集文字：1.2-1.5
- 轻松阅读：1.5-2.0

### 4. 护眼模式
夜间阅读建议：
- 主题：护眼
- 字体：18px+
- 行间距：1.5-2.0
